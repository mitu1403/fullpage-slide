Setting Up Tailwind CSS In A React Project

Getting Started

01 To set up our project, we’ll first create a new React app using create-react-app

npx create-react-app react-tailwind-project && cd react-tailwindcss

02 Next, we'll install a few development dependencies

npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p

This commands generate a tailwind.config.js and a postcss.config.js in the project’s base directory.

Configure your template paths

03 Add the paths to all of your template files in your tailwind.config.js file. 

module.exports = { 

    content: [
        "./src/**/*.{js,jsx,ts,tsx}",
    ],
    theme: {
        extend: {},
    },
    plugins: [], 
}

Add the Tailwind directives to your CSS

04 Add the @tailwind directives for each of Tailwind’s layers to your ./src/index.css file.

@tailwind base;
@tailwind components;
@tailwind utilities;

05 Run your build process

npm run start
